# Modern Data Warehouse Project (SQL Server)

## Overview

This project demonstrates the design and implementation of a modern Data Warehouse using Microsoft SQL Server. It follows the Medallion Architecture (Bronze, Silver, Gold) to ingest, clean, transform, and model data from multiple business systems into a centralized analytical warehouse.
The project focuses on building a scalable and well-structured data warehouse that supports business intelligence, reporting, and data analysis.

## Objectives

- Build a modern Data Warehouse using SQL Server.
- Integrate data from multiple source systems.
- Clean and standardize raw data before analysis.
- Design a business-friendly dimensional data model.
- Generate analytical datasets for reporting and decision-making.
- Follow industry-standard Data Warehouse design practices.

## Project Specifications

- Import data from multiple source systems (CRM & ERP).
- Load raw data into the Bronze Layer.
- Perform data cleansing and transformations in the Silver Layer.
- Create business-ready Dimension and Fact tables in the Gold Layer.
- Use SQL-based ETL processes for data loading.
- Apply consistent naming conventions and documentation.
- Focus on the latest available data (no historical tracking).
